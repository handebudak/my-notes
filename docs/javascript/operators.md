# Operators (Operatörler)

**Aritmetik Operatörler**:

- **`;`**   :Toplama operatörü
- **`-`**   : Çıkarma operatörü
- **`x`**   : Çarpma operatörü
- **`/`**   : Bölme operatörü
- **`%`**   : Mod (kalan) operatörü

**Atama Operatörleri**:

- **`=`** : Değer atama operatörü
- **`+=`** : Toplama atama operatörü
- **`=`** : Çıkarma atama operatörü
- **`=`** : Çarpma atama operatörü
- **`/=`** : Bölme atama operatörü
- **`%=`** : Mod atama operatörü

**Karşılaştırma Operatörleri**:

- **`==`** : Eşit mi?
- **`===`** : Kesinlikle eşit mi?
- **`!=`** : Eşit değil mi?
- **`!==`** : Kesinlikle eşit değil mi?
- **`>`** : Büyük mü?
- **`<`** : Küçük mü?
- **`>=`** : Büyük veya eşit mi?
- **`<=`** : Küçük veya eşit mi?

**Mantıksal Operatörler**:

- **`&&`** : Ve (and) operatörü
- **`||`** : Veya (or) operatörü
- **`!`** : Değil (not) operatörü
